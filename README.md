**Resume Builder** is a modern, design-centric application for creating professional resumes. It focuses on pixel-perfect layouts, modern typography, and interactive design controls that let developers build resumes that look like custom-designed portfolios.

---

## 🎨 Creative Features

- **🎭 Modern Templates**: 10+ professional styles including Minimal, Industrial, and Cinematic.
- **🖼️ Image FX**: Grayscale-to-color hover effects and custom border radius controls.
- **📊 Skill Bars**: Adjustable knowledge levels for every skill with toggleable visualization.
- **🛠️ Live Preview**: Real-time rendering as you type your experience and details.
- **💾 Auto-Save**: Persistent state management so your data is never lost during editing.

## 📂 Project Structure

```text
resume-builder/
```

## 🌈 Design Philosophy

This project prioritizes **Visual Excellence**. Unlike generic builders, it uses curated color palettes (HSL-based), Google Fonts (Inter/Outfit), and subtle micro-animations to ensure the output feels premium and state-of-the-art.
